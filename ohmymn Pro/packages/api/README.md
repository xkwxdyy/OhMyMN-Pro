Type definitions for the latest MarginNote API, and some utility functions that allow you to develop more powerful addon.

## Usage

```shell
pnpm add marginnote
# or
npm install marginnote
# or
yarn add marginnote
```

then

```ts
import { MN } from "marginnote"
```
